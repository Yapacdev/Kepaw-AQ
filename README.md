# KepawAQ

Prevent Windows from going to sleep or turning off the display using PowerShell.

---

## Features

- Blocks sleep mode
- Blocks display turn-off
- Lightweight and simple
- No installation required
- Automatically restores default power behavior on exit

---

## Usage

##### Run the kepawAQ_Luncher.bat and make sure the KepawAQ.ps1 exists in the same folder

##### Type Ctrl+C to stop the Script
